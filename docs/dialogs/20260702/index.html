■ 今回 7/2 の話題：CI構想と本荘論文HTML化
CI構想（Collective Intelligence）
  ・ **CI構想の定義**
    ・ 個人の発信 → AIの整理 → 他者の質問 
      → AIの応答 → 個人の理解 → 再発信
    ・ この循環を通じて、個人の知識を深め、
      集団の知性を高める

  ・ **CI vs CS**
    ・ **CI (Collective Intelligence)**
      集団の知性を高める
    ・ **CS (Collective Stupidity)**
      個人の知識が浅いまま、
      集団の愚かさを増幅

  ・ **知識循環の可視化**
CI構想の知識循環モデル
CI構想の実践例：本荘論文HTML化
  ・ **本荘論文HTML化におけるCI循環**
    1. **個人の発信**
       本荘先生の論文（専門的研究成果）
    2. **AIの整理**
       PDF抽出、HTML構造化、
       グラフ・表の視覚化
    3. **他者への還元**
       読者が理解しやすい形式で公開
    4. **個人の理解の深化**
       インタラクティブなグラフ・表で
       視覚的理解
    5. **再発信**
       引用・議論・新たな研究の基盤に

  ・ **作業内容**
    ・ PDFから全文をテキスト抽出
    ・ HTML構造化（セクション区切り、
      数式MathJax対応、参考文献整形）
    ・ インタラクティブグラフの追加
      （Chart.js、ダークモード対応）
    ・ 折りたたみ可能な詳細表
      （<details>タグで実装）

  ・ **HTML化のメリット**
    ・ 視覚的理解の向上
      グラフで一目で傾向がわかる
    ・ インタラクティブ性
      マウスオーバーで詳細値表示
    ・ 情報の階層化
      グラフ（概要）→ 表（詳細）
    ・ アクセシビリティ
      モバイル対応、スクリーンリーダー対応
    ・ リンク・参照の利便性
      セクション間ジャンプ、外部リンク

  ・ **技術的工夫**
    ・ グラフの配色
      GDP: 緑、失業率: 赤、物価: 青
    ・ ダークモード対応
      背景 #1a1a1a、テキスト #e0e0e0
    ・ データの整合性
      22年分×7列を手作業確認

  ・ **作成されたファイル**
    ・ honjo-2019-full.html
      フル版HTML
    ・ scripts/convert_honjo_2019.py
      メイン変換スクリプト
    ・ scripts/convert_table1_to_html.py
      表-1データ変換スクリプト

  ・ **CI循環における意義**
    ・ 視覚化による理解促進
      = 「他者への還元」の質を高める
    ・ 構造化されたデータ
      = 「AIの整理」が将来の学習に使える
    ・ 再利用可能な形式
      = 次の研究者が「再発信」しやすい基盤
    ・ **単なる技術作業ではなく、
      知識循環を促進する実践**
