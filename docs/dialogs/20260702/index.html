■ 今回 7/2 の話題：本荘 2019 論文のHTML化


  ・ 背景
    ・ 本荘 2019 論文（要約版）は既にHTML化されていた
    ・ しかし、図表が含まれていないため、データの詳細が見えなかった
    ・ 特に「表-1：主要経済指標の推移（1996-2017年度）」は重要なデータ

  ・ 今回の作業：フル版HTML化
    ・ PDFから全文をテキスト抽出
      ・ Python pypdf2 を使用
      ・ 抽出結果を honjo-2019-extracted.txt に保存
    ・ HTML構造化
      ・ セクション区切り（目次からのアンカーリンク）
      ・ 数式のMathJax対応
      ・ 参考文献の整形
    ・ インタラクティブグラフの追加
      ・ Chart.js を使用
      ・ 実質GDP成長率、完全失業率、消費者物価指数上昇率の3指標を可視化
      ・ ダークモード対応
    ・ 折りたたみ可能な詳細表
      ・ <details> タグで実装
      ・ デフォルトは閉じた状態（グラフを先に見せる）
      ・ 「📊 表-1：主要経済指標の詳細データを表示」をクリックで展開
    ・ トップページの更新
      ・ 要約版・フル版・PDFへのリンクを並列表示
      ・ ユーザーが用途に応じて選択できる構成

  ・ HTML化のメリット（PDFとの比較）
    ・ **視覚的理解の向上**
      ・ グラフで一目で傾向がわかる
      ・ 色分け・凡例で3指標の関係が把握しやすい
    ・ **インタラクティブ性**
      ・ グラフにマウスを乗せると詳細値が表示
      ・ 凡例クリックで指標の表示/非表示を切り替え可能
    ・ **情報の階層化**
      ・ グラフ（概要）を先に見せる
      ・ 詳細データ（表）は必要に応じて展開
      ・ ページの読み込みが軽い
    ・ **アクセシビリティ**
      ・ モバイル対応（レスポンシブ）
      ・ スクリーンリーダー対応
      ・ テキスト選択・コピーが容易
    ・ **リンク・参照の利便性**
      ・ セクション間のアンカーリンク
      ・ 外部リンク（出所：内閣府、総務省統計局など）

  ・ 作成されたファイル
    ・ honjo-2019-full.html - フル版HTML
    ・ honjo-2019-extracted.txt - PDF抽出テキスト（作業用）
    ・ scripts/convert_honjo_2019.py - メイン変換スクリプト
    ・ scripts/convert_table1_to_html.py - 表-1データ変換スクリプト
    ・ scripts/add_cross_references.py - 相互参照追加スクリプト

  ・ 技術的工夫
    ・ グラフの配色
      ・ GDP成長率：緑（#4CAF50）- 経済成長を示すポジティブな印象
      ・ 完全失業率：赤（#F44336）- 注意を引く色
      ・ 消費者物価指数：青（#2196F3）- 中立的な印象
    ・ ダークモード対応
      ・ グラフの背景色：#1a1a1a
      ・ テキスト色：#e0e0e0
      ・ グリッド線：半透明の白
    ・ データの整合性
      ・ PDFの表データを手作業で確認・転記
      ・ 合計22年分（1996-2017年度）× 7列のデータ
      ・ 出所リンクも元論文のまま記載
