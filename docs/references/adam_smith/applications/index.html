■ 目次


このページでは、アダム・スミスの『道徳感情論』の概念を、現代の具体的な問題に適用して考察します。
理論を実践的に理解し、「正確な理解」に基づく判断力を養うことを目的としています。

【コミュニティレベルの倫理問題】
  1. 自治会のゴミ集積所運営
     - 非会員の利用料金は適切か？
     - 無断利用への対応
     - 共感、正義、公平な観察者の視点

  2. 自治会役員の引受問題（作成予定）
     - なぜ役員を引き受けたくないのか？
     - 公共心と自己利益のバランス
     - コミュニティへの責任

【国際社会の倫理問題】
  3. ロシアのウクライナ侵攻（作成予定）
     - 主権侵害と国際法
     - プーチンの道徳的判断の欠如
     - 国際社会における「公平な観察者」は可能か？

  4. 中国の権威主義体制（作成予定）
     - ウイグル、チベット、香港への抑圧
     - 習近平政権の倫理的問題
     - 一党独裁と道徳感情

  5. イスラエル・パレスチナ問題（作成予定）
     - 歴史的経緯と現代の暴力
     - 両者の「共感」の不在
     - 正義と報復の境界

【経済と倫理の問題】
  6. 正規雇用と非正規雇用（作成予定）
     - 格差の道徳的意味
     - 企業の社会的責任
     - 「国富論」と「道徳感情論」の統合的理解

  7. 気候変動対策の失敗（作成予定）
     - 集団的愚かさ（CS）の事例
     - 世代間の公平性
     - 短期的利益と長期的視点

【民主主義と判断力】
  8. 愚かな指導者の選出を防ぐ（作成予定）
     - 有権者の道徳的判断力
     - メディアリテラシーと「正確な理解」
     - 「公平な観察者」の視点をいかに育むか

■ 応用例の学習方法


【各事例の構成】
各応用例のページは、以下の構成で作成されています：

  1. 問題の説明
     - 具体的な状況の記述
     - 関係者と利害の整理
     - 何が倫理的問題なのか

  2. スミスの視点からの分析
     - 該当する概念の適用（共感、公平な観察者、正義、善行など）
     - 理論がどう問題を照らすか
     - スミス自身ならどう考えるか

  3. 他の思想家の視点
     - ロールズ、カント、アリストテレスなど
     - 多角的な理解

  4. 実践的な対策・提案
     - 理論から導かれる具体的行動
     - 制度設計のヒント
     - 個人としてできること

  5. 関連する概念へのリンク
     - メインページの該当箇所
     - 他の応用事例との関連

【学習の進め方】
  ・ まず理論（メインページ）で基本概念を理解
  ・ 次に、関心のある具体例で理論を適用
  ・ 複数の事例を比較して、理解を深化

【重要な原則】
  ・ 理論を「知る」だけでなく、「使える」ようにする
  ・ 一つの事例を深く掘り下げる
  ・ 自分の言葉で説明できるようにする
  ・ 他の人と対話しながら理解を確認する
